GIT cheat sheet

Create

Clone an existing repository
$ git clone ssh://userygdomain.com/repo.git
Create a new local repository
$ git init

Local changes

Changed files in your working directory
$ git status
Changes to tracked files
$ git diff
Add all current changes to the next commit
$ git add .

Branches & tags

List all existing branches
$ git branch -av
Switch HEAD branch
$ git checkout <branch>
Create new branch based on your current HEAD
$ git branch <new-branch>
Create new tracking brach based on a remote branch
$ git checkout --track <remote/branch>
Delete a local branch
$ git branch -d <branch>

Merge & rebase

Merge <branch> into your current HEAD
$ git merge <branch>
Rebase your current HEAD onto <branch> (don't rebase published commits!)
$ git rebase <branch>
Abort a rebase
$ git rebase --abort
Continue a rebase after resolving conflicts
$ git rebase --continue
Use your configured merge tool to solve conflicts
$ git mergetool
